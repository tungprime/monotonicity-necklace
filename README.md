
# Monotonicity of necklace polynomials 

This Git repository contains the code we wrote in order to generate and conduct experiments necklace polynomials, as discussed in our paper:

**[1]** Sunil Chebolu, Ján Mináč, Tung T. Nguyen, Nguyen Duy Tan *Monotonicity of necklace polynomials* (Preprint 2026).

## Description

This repository provides functions for computing necklace polynomials. We use SageMath and some supporting Python 
libraries including NumPy and Matplotlib. It includes scripts to generate and visualize polynomial values; 
the resulting plots both validate our computations and helped identify promising research questions.
